JAVA 회원관리 데이터베이스 프로그램 구현





3학년 1학기 JAVA수업을 들으면서 만든 회원관리 데이터베이스 프로그램입니다.

초기 데이터(root 계정)를 이용하여 로그인을 한 후, 0번 메뉴에 있는 회원관리 프로그램을 실행하도록 구조가 이루어져 있습니다.



또한, 단순 과제와 달리 로그인 후 로그아웃 기능을 넣음으로써 프로그램 실행 중 중복성 위배(중복 로그인)을 방지할 수 있도록 구현해 보았습니다.

퀸-맥클러스키(Quine_McClusky) 프로그램 구현





2학년 1학기 C++언어를 들으면서 구현해 보았던 맥클러스키 프로그램입니다.

일반적으로 논리식을 구할 때 카르노 맵을 많이 사용하나, 5변수 이상(e 이상의 논리식)의 경우에는 단순 카르노맵을 사용하기 보다는 퀸-맥클러스키 방법을 사용해 간략화하는 방식을 취하고 있습니다.
